# Ekaacc Documentation

This project contains the documentation for the Ekaacc monorepo.

## Tech Stack

- [Next.js](https://nextjs.org/)
- [Nextra](https://nextra.site/)
- [React](https://reactjs.org/)
- [Tailwind CSS](https://tailwindcss.com/)

## Getting Started

1.  Install dependencies:
    ```bash
    npm install
    ```

2.  Run the development server:
    ```bash
    npm run dev
    ```

3.  Open [http://localhost:3001](http://localhost:3001) to view the docs.

## Structure

- `pages/`: Contains the documentation content (MDX files).
- `theme.config.jsx`: Theme configuration.
- `next.config.js`: Next.js configuration.
