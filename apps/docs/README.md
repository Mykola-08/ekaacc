# Ekaacc Documentation Portal

This is the documentation portal for the Ekaacc platform. It is a Next.js application.

## Features

- **Detailed Documentation**: Includes Getting Started, Tutorials, API Reference, and FAQ.

## Setup

1.  **Install Dependencies**:
    ```bash
    npm install
    ```

2.  **Run the App**:
    ```bash
    npm run dev
    ```

## Project Structure

- `src/app`: Next.js App Router pages.
- `src/components`: Reusable components.
