⚠️ Nota importante — oklch
El preset Nova usa oklch() en lugar de HSL. oklch funciona en Safari 15.4+, Chrome 111+, Firefox 113+. En Tailwind CSS v4 ya está soportado nativamente. Si necesitas soporte legacy, usa @supports fallbacks.

Dark Mode Specifics
sidebar-primary en dark = AZUL

En light mode el sidebar usa el mismo primary (negro), pero en dark mode --sidebar-primary cambia a oklch(0.488 0.243 264.376) — un azul. Tenlo en cuenta para el active state del nav.

accent en dark es más prominente

En dark --accent es oklch(0.371 0 0) mientras que en light es 0.97. Esto hace que los hovers sean más visibles en dark sin necesitar colores extra.

💡 menuAccent: subtle — qué significa
Con menuAccent: "subtle" el hover de items usa sidebar-accent (gris muy suave) en lugar de un color llamativo. El item activo usa sidebar-primary (negro en light, azul en dark). Esto da un look limpio y profesional.

12

      

        
Cómo aplicarlo a tu dashboard existente

        
Si ya tienes Supabase + Stripe + auth funcionando

      

    


    

      

        
Paso 1 — Reemplaza globals.css

        
Copia el CSS de tokens de la sección 2 en tu globals.css. Asegúrate que tienes los @import de tw-animate-css y shadcn/tailwind.css arriba de todo.

      

      

        
Paso 2 — Actualiza layout.tsx

        
Importa Geist desde geist/font/sans (no de Google Fonts). Aplica GeistSans.variable en el <html>. Wrap con SidebarProvider si usas sidebar.

      

      

        
Paso 3 — Instala HugeIcons

        
Reemplaza tus iconos actuales con @hugeicons/react. Todos los componentes shadcn Nova esperan HugeIcons. npm install @hugeicons/react @hugeicons/core-free-icons

      

      

        
Paso 4 — Dark mode con next-themes

        
Instala next-themes. Wrap con <ThemeProvider attribute="class">. El preset Nova usa la clase .dark en el html element para activar el dark mode.

      

    


    

      
Dark mode setup

      
npm install next-themes

// providers.tsx
"use client"
import { ThemeProvider } from "next-themes"

export function Providers({ children }) {
  return (
    <ThemeProvider
      attribute="class"
      defaultTheme="system"
      enableSystem
      disableTransitionOnChange
    >
      {children}
    </ThemeProvider>
  )
}

// Botón toggle
import { useTheme } from "next-themes"
const { theme, setTheme } = useTheme()
<Button onClick={() => setTheme(theme === "dark" ? "light" : "dark")}>
  Toggle
</Button>
