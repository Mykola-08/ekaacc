# **App Name**: EKA Account

## Core Features:

- Personalized Dashboard: Dynamic dashboard with key metrics, AI insights, and quick actions tailored to the user's role (User, Therapist, Donor, etc.).
- AI Assistant: Context-aware AI tool to answer questions, perform actions (book session, summarize reports), and suggest exercises.
- Session Management: Booking system, therapist tools (outcome entry, ratings, notes), client forms, AI trends, and milestone tracking.
- Donation Engine: Full donation engine with receiver mode (funded session tracker, progress roadmap) and donor view (impact summaries).
- Reporting and Messaging: Therapist and user reports, SOAP summaries, AI-generated summaries, internal messaging, and context-aware AI chat support.
- Billing and Account Management: Subscription management, linked accounts, profile preferences, consents, and data export.
- Dynamic Personalization: Adaptive quick actions, AI tone adjustment, and service suggestions based on user type and linked accounts.

## Style Guidelines:

- Primary color: Soft blue (#74b9ff) to evoke a sense of trust, professionalism, and calm.
- Background color: Light gray (#f5f5f5), near white, creating a clean, minimalist base.
- Accent color: Slightly darker blue (#4a90e2) used for interactive elements and highlights, providing good contrast against the background.
- Body and headline font: 'Inter', a sans-serif font with a modern and neutral appearance suitable for both headlines and body text.
- Simple, clean icons with rounded corners, following the minimalist design. Icons should be visually consistent with the chosen font.
- Apple-inspired layout with white space, soft gray panels, rounded corners (16-20px), and subtle shadows for a modern, breathable design.
- Subtle, smooth transitions and animations to enhance user experience without being distracting.
