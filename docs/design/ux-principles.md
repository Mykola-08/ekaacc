# UX Principles Library

This file captures the UX and behavioral design principles to apply across EKA
Balance. Updated every few weeks based on
[Growth.Design Psychology](https://growth.design/psychology).

## How to use this file

- Use these principles to review new pages before merge.
- Prioritize principles that reduce friction, increase clarity, and improve
  trust.
- Do not apply all principles at once on one screen.
- Pick 3-5 principles per flow and validate with real usage data.

## Practical baseline applied in this repository

- Clear focus indicators and keyboard navigation support.
- Skip-to-content entry point for faster access.
- Touch-friendly action target sizing for key CTAs.
- Main content landmarks across marketing, dashboard, and platform layouts.
- Progressive disclosure through grouped navigation and staged flows.
- Strong visual hierarchy in dashboard and marketing headers.
