# Setup

This page centralizes quick setup instructions and links to detailed guides.

- **Quick Start / Local Dev**: See the main project `README.md` for step-by-step local development: `../README.md`.
- **Database Setup**: Follow the full instructions in `DATABASE_SETUP_GUIDE.md`.
- **Environment**: Copy `.env.example` to `.env` and configure Supabase, Stripe, and any OAuth providers.
- **Install**:

```powershell
npm install
npm run dev
```

- **Useful links**:
  - `DATABASE_SETUP_GUIDE.md` - Database provisioning and migrations
  - `PRODUCTION_DEPLOYMENT_GUIDE.md` - Production deployment information
  - `TESTING.md` - How to run the tests
